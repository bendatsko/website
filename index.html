My research focuses on high-performance mixed-signal integrated circuits, edge computing, and embedded systems. These entries are marked with an asterisk (*) and will be updated to reflect latest developments/publications as they become available.

*An 8-bit 20.7 TOPS/W Multi-Level Cell ReRAM-based Compute Engine
Justin M. Correll, Lu Jie, Seungheun Song, Seungjong Lee, Junkang Zhu, Wei Tang, Luke Wormald, Jack Erhardt, Nicolas Breil, Roger Quon, Deepak Kamalanathan, Siddarth Krishnan, Michael Chudzik, Zhengya Zhang, Wei D. Lu, Michael P. Flynn
2022 IEEE Symposium on VLSI Technology and Circuits (VLSI Technology and Circuits)

An ASIC designed for accelerating scientific computing and machine learning in edge computing contexts. Integrates self-contained ReRAM-based compute-in-memory (CIM) tiles with a RISC-V host, delivering high efficiency and compute density.

*A 16-Element Fully Integrated 28GHz Digital Beamformer with In-Package 4x4 Patch Antenna Array and 64 Continuous-Time Band-Pass Delta-Sigma Sub-ADCs
Rundao Lu, Christine Weston, Daniel Weyer, Fred Buhler, Michael P. Flynn
2020 IEEE Radio Frequency Integrated Circuits Symposium (RFIC)

A 28GHz digital beamformer integrated circuit featuring a 4x4 patch antenna array capable of generating multiple beams simultaneously for mm-wave communication. This technology assists in advancing modern cell-phone communication, radar systems, and allows emergency response drones to offer more reliable internet to disaster-stricken regions.

Modulated Signal Source from ADI Components for Versatile Frequency Generation from 20GHz to 40GHz
Hao Yu, Christine Weston, Abraham Lopez, Benjamin Datsko, Michael P. Flynn

A signal generation system incorporating the ADS7-V2EBZ FPGA, AD9174 FMC EBZ high-frequency RF DAC, and ADMV1013 high-frequency broadband up converter. This setup utilizes a MATLAB function to generate "circular" signal sequences, allowing for smooth replay by the FPGA without discontinuities. Achieves 5% Error Vector Magnitude (EVM) for 64-QAM and 1% EVM for 16-QAM and single-tone sequences.

Ultra-Low Noise 5V Power Supply for Noise-Sensitive IC Testing with Dual 18650 LiPo Battery Cells and Sub-3mV Pk-Pk Ripple
Benjamin Datsko, Luke Wormald, Michael P. Flynn

A 5V power supply designed for noise-sensitive integrated circuit testing, utilizing 18650 LiPo battery cells, ferrite beads, and low-pass filters to eliminate high-frequency switching noise. Approximately 8.4V from USB or batteries feed into a high-PSSR 5V LDO, reducing output ripple and ensuring a stable and clean power source. Uses an ATmega32U4 chip and runs an RTOS programmed in C++ leveraging PlatformIO for BMS and display control.


Projects
AiApply: Create Tailored Resumes Using Generative AI

A web app for tailoring resumes to job postings based on previous work experience by gathering user's work history and generating tailored, ATS-friendly LaTeX resumes for each user-provided job posting. Features React front-end, Flask backend, OAuth for authentication, and Firebase Firestore; deployed on AWS EC2.

WebSim: Michigan Solar Car Race Simulator Microservice

A navigable hub for the U-M Solar Car Team's crucial simulators and race strategy tools. Built on client-server architecture featuring React Native front-end, CRUD API written in EJS for accessing MongoDB simulation storage database, and Flask simulator API for a one-size fits all means of invoking simulators written in various languages (Java, C, etc.) via the subprocess method and piping output to a centralized location. Hosted on AWS EC2, served using Nginx; PM2 used for container orchestration, and Certbot 2 used for domain management.

Quickkey: A Minimalistic Typing Test

A modern and minimalistic website designed to help evaluate and improve typing speed and accuracy. Developed using Typescript and React. Prototyped in Figma.

Upper-Atmospheric Weather Balloon Payload System

A high-altitude Arduino-based payload system equipped with a TMP36 temperature sensor, thermistor, pressure sensor, humidity sensor, accelerometer, UV sensor, and GPS for the purpose of analyzing the state of the ozone layer.

PathVis: VanillaJS Pathfinding Algorithm Visualizer

A pathfinding algorithm visualizer with the capacity to visualize a variety of algorithms, including Dijkstra's, A-Star, Greedy BFS, and Greedy DFS. Written in Vanilla JavaScript. Prototyped in Adobe XD and Adobe Photoshop. Static site hosted by Netlify.

TempoTimer: Race Race Preparation for Swimmers

A mobile application that functions as an underwater metronome for swimmers' pre-race visualizations. It combinesn auditory cues with mental imagery, ensuring each stroke visualized by the athlete is in sync with the the rhythm set within the app, and thus their goal time. Developed in Java using the Android SDK. Available on the Google Play Store.
