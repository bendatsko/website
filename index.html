Research

I frequently work within the realms of mixed-signal integrated circuits, edge computing, PCB design, and embedded software. Some projects haven't been published on for a long while but are still under development. Representative titles are marked with a star *.

*An 8-bit 20.7 TOPS/W Multi-Level Cell ReRAM-based Compute Engine
Justin M. Correll, Lu Jie, Seungheun Song, Seungjong Lee, Junkang Zhu, Wei Tang, Luke Wormald, Jack Erhardt, Nicolas Breil, Roger Quon, Deepak Kamalanathan, Siddarth Krishnan, Michael Chudzik, Zhengya Zhang, Wei D. Lu, Michael P. Flynn
2022 IEEE Symposium on VLSI Technology and Circuits (VLSI Technology and Circuits)

An ASIC designed for accelerating scientific computing and machine learning in edge computing and IoT devices. Integrates self-contained ReRAM-based compute-in-memory (CIM) tiles with a RISC-V host, delivering high efficiency and compute density.

*A 16-Element Fully Integrated 28GHz Digital Beamformer with In-Package 4x4 Patch Antenna Array and 64 Continuous-Time Band-Pass Delta-Sigma Sub-ADCs
Rundao Lu, Christine Weston, Daniel Weyer, Fred Buhler, Michael P. Flynn
2020 IEEE Radio Frequency Integrated Circuits Symposium (RFIC)

A 28GHz digital beamformer integrated circuit featuring a 4x4 patch antenna array capable of generating multiple beams simultaneously for mm-wave communication. Beamforming is what allows signals to be "pointed" different directions in space.

Modulated Signal Source from ADI Components for Versatile Frequency Generation from 20GHz to 40GHz
Hao Yu, Christine Weston, Abraham Lopez, Benjamin Datsko, Michael P. Flynn

A signal generation system incorporating an FPGA, high-frequency RF DAC, and high-frequency broadband up converter. A MATLAB script generates "circular" signal sequences (16-QAM, 64-QAM, single-tone, etc.) which are looped by the FPGA, converted to analog, and up-converted from 100 MHz to radio-frequency (28 GHz).

Ultra-Low Noise 5V Power Supply for Noise-Sensitive Testing Leveraging LiPo Batteries
Benjamin Datsko, Luke Wormald, Michael P. Flynn

A 5V power supply designed for noise-sensitive integrated circuit testing, utilizing 18650 LiPo battery cells, ferrite beads, and low-pass filters to eliminate high-frequency switching noise. Approximately 8.4V from USB or batteries feed into a high-PSSR 5V LDO, reducing output ripple to ~3mV Pk-Pk and ensuring a stable signal. Uses an ATmega32U4 RISC-based microcontroller to run a C++ RTOS for scheduling BMS, sensor read-in, and menu processes using the PlatformIO development environment.

Projects
AiApply: Create Tailored Resumes With Generative AI

Help tailor your resume to job postings provided a master resume! Features a React front-end, Flask endpoints for LaTeX-to-pdf conversion, OAuth-based authentication, and uses Firebase for cloud storage.

University of Michigan Solar Car Team Race Simulator Dashboard and Microservice

A web platform for the U-M Solar Car Team's race physics and aerodynamics simulators. Features a React front-end, a MongoDB collection for simulation data, and a Flask app containing an API with interfaces to each simulator.

Upper-Atmospheric Weather Balloon Payload System

A high-altitude Arduino-based payload system (components: weather balloon, case, and PCB) equipped with a TMP36 temperature sensor, thermistor, pressure sensor, humidity sensor, accelerometer, UV sensor, GPS, OpenLog, and MicroSD writer for recording and analyzing upper-atmospheric conditions.

Quickkey: A Minimalistic Typing Test

A simplistic typing speed test. Speed measured in words per minute. Developed using Typescript, React, and NodeJS.

PathVis: A Pathfinding Algorithm Visualizer

A visualizer for search algorithms including Dijkstra's, A-Star Search, Greedy BFS, and Greedy DFS written in vanilla JavaScript. Deployed to Netlify.

TempoTimer: Race Pace Android App for Swimmers

Visualizing success is popular in sports psychology. This app acts as metronome for swimmers' strokes and provides an auditory simulation of swimmers' races based on inputted splits which can be played during visualization to increase mental confidence. Developed in Java using the Android SDK. Available on the Google Play Store.
